# Codigo aislado de inventario

Esta carpeta conserva codigo que ya no forma parte del build principal de la
app de servicios de construccion.

Contenido:

- `M_Articulos.prg`: maestro completo de articulos, precios, stock y cuentas.
- `M_Familias.prg`: mantenimiento de familias de articulos.
- `Informes_Inventario.prg`: informes de articulos y stock minimo.

No se borra el trabajo: queda disponible para reutilizarlo en otra app con
inventario, almacen o movimientos de mercancias.
