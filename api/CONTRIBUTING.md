# Contributing to GfxStack API

The `api/` directory is intended to remain reusable free code.

When you improve, fix, or extend API files, please send the changes back to the
original GfxStack API repository as a patch or pull request whenever practical.

License rules:

- API files are licensed under MPL-2.0.
- Modified API files distributed to others must remain under MPL-2.0.
- Applications that merely use the API do not need to adopt MPL-2.0 for their
  own separate source files.

Recommended file header for API source files:

```harbour
/*
 * SPDX-License-Identifier: MPL-2.0
 * Copyright (c) 2026 Oswaldo Fernando
 */
```
